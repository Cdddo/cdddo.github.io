  • A header with the company logo
  • A big email title that is a call to action
  • A Message box with a short message
  • A services box with a title and a few icons with captions
  • An image grid to show possible uses or projects
  • A call to action button
  • A footer with the contact info

We should not forget:

  • CSS cannot use an external stylesheet
  • All image links must be absolute

  • Test on a lot of email clients and mobile devices
